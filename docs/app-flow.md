# App Flow

## Start

The app opens a desktop window and displays the simulated account balance.

## Deposit

The user enters a positive amount and selects deposit. The app validates the amount, updates the simulated balance, and records the transaction in memory.

## Withdraw

The user enters a positive amount and selects withdraw. The app checks for sufficient funds before updating the simulated balance.

## Transaction History

The app keeps a simple in-memory history during the session. Future versions can save history to a local file or database.
